# KNX Sentinel Repository

This is a Home Assistant Add-on Repository containing the **KNX Sentinel** agent.

## How to Install

1. **Copy the Repository URL**:
   ```
   https://github.com/jangoachayan/Diagnostics-Sentinel
   ```

2. **Add to Home Assistant**:
   - Go to **Settings > Add-ons > Add-on Store**.
   - Click the **three dots** (top right) > **Repositories**.
   - Paste the URL and click **Add**.

3. **Install the Add-on**:
   - Find "KNX Sentinel" in the list (you may need to refresh/check for updates).
   - Click **Install**.

## Add-ons

- **[KNX Sentinel](knx-sentinel/README.md)**: High-fidelity distributed KNX monitoring agent.
